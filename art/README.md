# Art

AI image generation and editing using Google Gemini models (Nano Banana). Create images, edit existing images, and produce art assets.

## Prerequisites

- AI Launchpad marketplace added — see [main README](../README.md)
- `GEMINI_API_KEY` environment variable set with a [Google Gemini API](https://ai.google.dev/) key

## Installation

```
/plugin install art@ai-launchpad-marketplace
```

Restart Claude Code for the changes to take effect.

## Skills

| Skill | Description |
|-------|-------------|
| `nanobanana` | Text-to-image generation, image editing, multi-image references, 1K/2K/4K resolution |
