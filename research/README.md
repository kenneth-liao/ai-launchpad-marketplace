# Research

Web research capabilities for competitor analysis, market research, and general information gathering. Uses Claude Code's built-in web search — no additional API keys required.

## Prerequisites

- AI Launchpad marketplace added — see [main README](../README.md)

## Installation

```
/plugin install research@ai-launchpad-marketplace
```

Restart Claude Code for the changes to take effect.

## Skills

| Skill | Description |
|-------|-------------|
| `business-competitor-analysis` | Comprehensive competitor analysis with market positioning, pricing, features, SWOT, and competitive matrix |

## Agents

| Agent | Description |
|-------|-------------|
| Web Researcher | General research agent for web search, data gathering, and synthesizing insights |
