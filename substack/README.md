# Substack

Orchestrator plugin for Substack content workflows — newsletter issues and Substack Notes. Composes foundation skills from `content-strategy`, `writing`, and `visual-design` into Substack-specific production pipelines.

## Prerequisites

- AI Launchpad marketplace added — see [main README](../README.md)

## Installation

```
/plugin install substack@ai-launchpad-marketplace
```

Restart Claude Code for the changes to take effect.

## Skills

| Skill | Description |
|-------|-------------|
| `plan-issue` | Plan a complete newsletter issue from topic to publication-ready content |
| `optimize-issue` | Optimize an existing draft or write a full issue from an outline |
| `create-note` | Create standalone Substack Notes for engagement and audience growth |
