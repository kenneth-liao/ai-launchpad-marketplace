# Branding Kit

Create and manage visual design systems and brand guidelines. Define brand aesthetics, generate style guides, and produce consistent visual assets.

## Prerequisites

- AI Launchpad marketplace added — see [main README](../README.md)

## Installation

```
/plugin install branding-kit@ai-launchpad-marketplace
```

Restart Claude Code for the changes to take effect.

## Skills

| Skill | Description |
|-------|-------------|
| `design-system` | Create a complete visual design system with color palette, typography, and style board |
| `brand-guidelines` | Define brand identity and generate an enforceable brand guidelines skill |
