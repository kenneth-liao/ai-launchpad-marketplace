# Writing

Writing content in Kenny Liao's authentic voice across all content types and platforms — newsletters, YouTube scripts, guides, sales copy, social posts, and more.

## Prerequisites

- AI Launchpad marketplace added — see [main README](../README.md)

## Installation

```
/plugin install writing@ai-launchpad-marketplace
```

Restart Claude Code for the changes to take effect.

## Skills

| Skill | Description |
|-------|-------------|
| `copywriting` | Generalized writing for any platform and format — handles structure, drafting, and platform-specific conventions |
| `voice` | Applies Kenny's authentic writing voice — invoked automatically by copywriting or independently by other skills |
